## Description

Test Tech - Backend Developer
This is a technical test for the Backend Developer position at Payever. The objective of this test is to evaluate my knowledge in the field of backend development.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

## Prerequisites

To run this project, you will need to have Node.js and npm installed on your machine. You can download them from the official website: https://nodejs.org/en/download/. You will also need Nest.Js - https://nestjs.com/

## API Endpoints

The following endpoints are available in the API:

- POST /users: Create a user and send him an email
- GET /users/:id: Returns information about a specific user
- DELETE /api/user/{userId}/avatar: Deletes a specific user
- GET /api/user/{userId}/avatar: Download image and save it in the database

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
